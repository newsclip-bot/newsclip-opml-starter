# newsclip-opml-starter

Generate a starter OPML file referencing Newsclip categories.

A practical helper for daily content operations.

## Why this exists
- Provide a simple starting point that can be expanded into a fuller tool.
- Provide a simple starting point that can be expanded into a fuller tool.

## Project references
- [Newsclip](https://newsclip.com/)
- [Newsclip Business](https://newsclip.com/business/)
- [Newsclip Sports](https://newsclip.com/sports/)

## Usage
Basic usage:

```bash
npm install
npm run start -- https://newsclip.com/
```

## Notes
- Treat output as advisory, not a strict source of truth.
- For production, you can swap the parsing layer with a robust library.
